TaskFlow
- Osobisty projekt frontendowy będący częścią mojej nauki programowania. TaskFlow to prosta aplikacja do zarządzania zadaniami w stylu Kanban.
Funkcjonalności

Tworzenie zadań z tytułem i opisem
- Podgląd szczegółów zadania w oknie modalnym
- Przeciąganie zadań między statusami (Oczekujące / W trakcie / Skończone) za pomocą Drag & Drop
- Licznik zadań w każdej kolumnie
- Zapisywanie zadań w localStorage (dane nie znikają po odświeżeniu)
- Responsywny interfejs (desktop, tablet, mobile)

Technologie
- HTML
- CSS
- JavaScript

Uruchomienie
- Pobierz pliki projektu i otwórz plik TaskFlow.html w przeglądarce.
